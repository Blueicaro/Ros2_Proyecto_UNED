Descripción del Proyecto

Este proyecto implementa un robot tipo "roomba" ROS 2. El objetivo es que el robot reciba comandos y registre colisiones.

El proyecto está en construcción. Según avance se irá acutalizando la documentación. El plazo de finalización es en Mayo de 2026

Ejecución y Resultados

Para ejecutar el proyecto, ejecuta el siguiente comando en la terminal:

ros2 launch robot_bringup_gz launch.xml

Resultados esperados:

  • El robot se mueve hacia adelante al recibir el comando de avance.
  • El robot se detiene al recibir el comando de detención.
  • El robot gira a la izquierda o derecha al recibir los comandos correspondientes.
  • El robot retrocede al recibir el comando de retroceso.
  • Los datos de los sensores se registran correctamente en la consola.

Para lanzar la aplicación en python que usar QT para crear una interfaz gráfica:

os2 run gui_py gui_py_node
