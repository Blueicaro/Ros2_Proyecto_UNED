Proceso de Modelado

El chasis del robot fue diseñado utilizando el banco de trabajo Part Design. Se priorizó la modularidad para facilitar el montaje de los sensores LiDAR y la cámara.

Figura 1: Diseño del chasis en FreeCAD.

De FreeCAD a ROS

Para integrar el diseño en ROS, se exportaron los modelos 3D en formato STL. Estos archivos se utilizaron para simular el robot en Gazebo. El archivo URDF se generó a partir de estos modelos para definir la estructura del robot en ROS.
