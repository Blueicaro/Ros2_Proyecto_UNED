Proceso de Modelado

El chasis del robot fue diseñado utilizando el banco de trabajo Part Design. Se priorizó la modularidad para facilitar el montaje de los sensores LiDAR y la cámara.

De FreeCAD a ROS

Para integrar el modelo en Gazebo/Rviz, utilicé el plugin FreeCAD-to-URDF. Es vital definir correctamente los links y joints (uniones) para que la cinemática funcione en ROS.
